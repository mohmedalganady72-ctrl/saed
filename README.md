# Saed App 🎓

**The First Student Platform for Exchanging Academic **

Saed App is a technical solution designed specifically for  students to help reduce the financial burden of academic supplies. The application acts as a secure intermediary that allows students to sell, buy, or donate books and academic tools (engineering, medical, etc.), promoting a culture of academic cooperation and solidarity.

---

## ✨ Features

- 🔐 User Login  
- 👤 Profile Setup  
- 📚 Browse academic supplies by category  
- 🔍 Search functionality  
- ➕ Add new item   
- 📄 View item details  
- ❤️ Promote donation culture among students  
- ⚡ Smooth user experience with optimized performance

## 🏗 Architecture (Clean Architecture)

The project follows Clean Architecture principles to ensure:

- Separation of concerns  
- Scalability  
- Maintainability  
- Testability  



## ⚙️ State Management

The project uses Cubit (Bloc Library) for state management, providing:

- 🔄 Unidirectional Data Flow  
- 🧠 Clear separation between UI and Business Logic  
- 🚀 Optimized rebuild performance  
- 📊 Predictable states (Initial, Loading, Loaded, Error)
